# WAN

The **WAN** panel displays the status of your [WAN](/glossary/w/wan). It also lets you configure [DNS](/glossary/d/dns) servers.

##

![WAN panel](./main.png)

^ Item ^ Description^
| Internet | Status of Internet connection. |
| Link | Status of link. |
| WAN IP(s) | IPv4 and [IPv6](/glossary/i/ip) address to the device. |
| Gateway(s) | IPv4 and [IPv6](/glossary/i/ip) address to [gateway](/glossary/g/gateway).  |
| Link Type | Ethernet |
| Link Speed | Auto-negotiated 1000 Mbps Full Duplex |
| DNS-Servers | IPv4 and IPV6 addresses to [DNS servers](/glossary/d/dns_server). |
| WAN uptime | Time since last disconnect for IPv4 and IPV6 WAN connection. |
