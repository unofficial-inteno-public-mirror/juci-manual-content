# Status

The **Status** tab shows information about the client and the connection.

## Status Information

^ Item ^ Description ^
| Hostname | The client [Hostname](/glossary/h/hostname). |
| IP Address | Assigned [IP](/glossary/i/ip) address. |
| MAC Address |[MAC](/glossary/m/mac) address. |
| DHCP | [DHCP](/glossary/d/dhcp) status. |
| Connected | Connection status. |
| Link Speed | Type of [negotiation](/glossary/a/auto_negotiation), [speed](/glossary/l/link_speed) and [duplex](/glossary/d/duplex) for the connection. |


## Wireless Details

For WiFi clients, the **Wireless Details** section shows detailed information about the wireless connection. All data is measured since last downtime.

^ Item ^ Description ^ Example ^
| Frequency | [WiFi frequency band](/glossary/w/wifi_band) for the access point. | 2.4GHz |
| RSSI |[RSSI](/glossary/r/rssi) strength for the signal. | -64 dBm |
| SNR | [Signal-To-Noise-Ratio](/glossary/s/snr). | 21 dBm |
| Idle | Time idle. | 1 s |
| In Network | Time in network. | 1813 s |
| WME | Status of [WMM](/glossary/w/wmm). | True |
| Power Save | Is Power save enabled? | False |
| N Mode | Is [802_11n](/glossary/0/802_11n) supported?  | True |
| VHT Mode | Is [802_11ac](/glossary/0/802_11ac) supported? | False |
| TX Bytes | Transmitted bytes. | 2438426 |
| RX Bytes | Transmitted bytes.  | 347988 |
| TX Rate | Transmission rate. | 58 Mbps |
| RX Rate | Recieve rate. | 6 Mbps |
