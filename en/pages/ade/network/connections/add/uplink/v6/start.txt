# DHCP v6 (Uplink)

An DHCP v6 connection uses an IPv6 address provided by a DHCP server.

### Step 1

In the first step you select basic settings for the interface.

^ Item ^ Description ^
| Interface Name | Name for the interface. |
| Interface Type | Select [interface protocol type](/glossary/i/interface_protocol_type). |


## Finalize

In the final step you select protocol, adapter and firewall settings for the interface.

^ Item ^ Description ^
| Protocol | Select [protocol](/glossary/p/protocol). |
| Interface Type | Select [interface protocol type](/glossary/i/interface_protocol_type). |
| Ethernet Adapter | [Base Device](../../../../devices/base/start) to create interface for. |
| Add network to a firewall zone | Connects interface to [firewall zone](../../../../firewall/zones/start). |
