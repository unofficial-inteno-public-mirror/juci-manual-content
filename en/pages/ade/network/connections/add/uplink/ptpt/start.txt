# Point-to-point Tunnel

A Point-to-Point Tunnel connection uses [PPP](/glossary/p/ppp)  across a [VPN](/glossary/v/vpn) tunnel to establish the network.

In the final step you select protocol and firewall settings for the interface.

^ Item ^ Description ^

| Add network to a firewall zone | Connects interface to [firewall zone](../../../../firewall/zones/start). |
