# VDSL

The **VSDL** view  allows you to configure [VDSL](/glossary/v/vdsl) devices.

##

At the top of the page is a list of selectable devices.

When a particular device is selected, details about it is shown in the configuration section.


^ Section ^ Description ^
| Name | Name of the device. |
| DSL Latency Path | DSL [Latency Path](/glossary/l/latency_path) 1, 2 or both 1 & 2. |
| PTM Priority| Normal or High [PTM Priority](/glossary/p/ptm_prio) . |
| IP QoS Schedule Algorithm | [Strict Priority Precedence](/glossary/s/spp) / [Weighted Fair Queuing](/glossary/w/wfq). |
| Bridge | Setting to enable [network bridge](/glossary/n/network_bridge) use. |
