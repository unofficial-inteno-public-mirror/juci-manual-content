# System Storage

The **System Storage Status** view shows information about [file systems](/glossary/f/file_system) and space used.

## Examples

^ Option ^ Description ^
| rootfs(/) | Root. |
| tmpfs(/tmp) | Temporary.  |
| tmpfs(/dev)| Devices. |
| tmpfs(/mnt)| Mount point. |
| tmpfs(/dev/sda1)| An attached USB stick. |
