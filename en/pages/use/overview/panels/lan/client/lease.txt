# Static Leases

The **Static Leases** tab allows you to assign a static [IP address](/glossary/i/ip_address) [dhcp lease](/glossary/d/dhcp_lease) to the client.


## Static Leases Section

^ Item ^ Description ^
| Device Name | [Hostname](/glossary/h/hostname) for [IPv4](/glossary/i/ipv4) |
| IP Address | IP address for [IPv4](/glossary/i/ipv4) |
| DUID | [DUID](/glossary/d/duid) for [IPv6](/glossary/i/ipv6) |
| Host ID | [Host ID](/glossary/h/hostid) for [IPv6](/glossary/i/ipv6) |

## Static Leases Settings

To assign a static address to the client:

* Click the {{/icon/add.png}} **add** button to open the section

* Add information for the type of network(s) you use
