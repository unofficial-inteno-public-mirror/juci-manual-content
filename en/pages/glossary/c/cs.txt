

# Class Selector

The Class Selector (CS) is used by [Differentiated Services](/glossary/d/diffserv) maps [Differentiated Services Code Point](/glossary/d/dscp) to indicate [precedence](/glossary/p/precedence).

It is backwards compatible with [IP precedence](/glossary/i/ip_precedence) values (CS0 = IP precedence 0, CS1 = IP precedence 1, and so on) .
