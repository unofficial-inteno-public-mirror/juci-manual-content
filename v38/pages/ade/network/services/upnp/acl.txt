# ACL

The **ACL** tab lets you configure the [Access Control List](/glossary/a/acl) for [UPNP](/glossary/u/upnp) access.

##

^ Item ^ Description  ^
| Comment | Description of the rule.  |
| External ports | External [ports](/glossary/p/port) to filter. |
| Internal addresses | Internal  addresses to filter. |
| Internal ports | Internal [ports](/glossary/p/port) to filter. |
| Action | `Allow` / `Deny` |
| Sort | Change order of list items. |
