# Forwarding

[Port Forwarding](/glossary/p/port_forwarding) allows remote computers to connect to a specific device within your private network.

##

The forwarding list shows information about any configured port forwarding rules.

^ Item ^ Comment ^
| Name | Identifier for the mapping. |
| Direction | [zone](/glossary/f/firewall_zone) involved |
| Dst. IP Address | Client [IP](/glossary/i/ip) address. |
| Protocol | Mapping [protocol](/glossary/p/port) ([UDP](/glossary/u/udp) / [TCP](/glossary/t/tcp) / TCP + UDP ). |
| Public port(s) | Public (external) [port](/glossary/p/port). |
| Private port(s) | Private (client) [port](/glossary/p/port). |


## Port Mapping Settings

To map incoming connections:

* Click the {{/icon/add.png}} **add** button to open the settings

The port mapping dialog lets you add configuration settings for the mapping.

Ports can be added one by one (`80`), as comma-separated lists (`8080, 8090`) or as ranges (`21-22`).

* Add information:

    * Add a name as identification

    * Add ports:

        * Add public/incoming port(s)

        * Add private/client port(s)

    * Select protocol

* Click **Save**

* Click **Close**

Your information is saved and is visible in the mapping list.
