# SIP Accounts

The **SIP Accounts** view shows information about configured [SIP accounts](/glossary/s/sip_account) for the device.

##

At the top of the page is a list of selectable accounts.

When a particular account is selected, details about it is shown in the configuration section.

^ Item ^ Description ^
| Enabled | Turn account on or off. | 
| Account Name | Name of [SIP account](/glossary/s/sip_account). |
| SIP domain name | Name of [SIP domain](/glossary/s/sip_domain). |
| SIP Username | The [SIP account username](/glossary/s/sip_user) for the account. |
| SIP Authentication Name | [SIP Authentication Name](/glossary/s/sip_auth) used with password to register with SIP server. |
| SIP Password | Enter new password to change. |
| Show Key Text | Display the password. |
| Display Name | Display name used in Caller ID. |
| SIP Server/Registrar | Address for [SIP server](/glossary/s/sip_server). |
| SIP Server/Registrar Port | [Port](/glossary/p/port) for [SIP server](/glossary/s/sip_server). |
| SIP Outbound Proxy | Address for outbound [proxy](/glossary/p/proxy). |
| SIP Outbound Proxy Port | [Port](/glossary/p/port) for outbound [proxy](/glossary/p/proxy). |
| Incoming Phone Lines | Check boxes for connected phone line ports. |
| Preferred codecs | Order of preference for [SIP codecs](/glossary/s/sip_codec). |
| G.711MuLaw Packetization | [Packetization](/glossary/p/packetization) setting for [G.711MuLaw](/glossary/s/sip_codec). |
| G.726 Packetization | [Packetization](/glossary/p/packetization) setting for [G.726](/glossary/s/sip_codec). |
| G.729a Packetization | [Packetization](/glossary/p/packetization) setting for [G.711ALaw](/glossary/s/sip_codec). |
| G.G.729a Packetization | [Packetization](/glossary/p/packetization) setting for [G.729a](/glossary/s/sip_codec). |
| Autoframing | Negotiate [packetization](/glossary/p/packetization) when call is established. |
| SIP Transport | [UDP](/glossary/u/udp) / [TCP](/glossary/t/tcp) / [TLS](/glossary/t/tls) |
| Encryption | Use [Secure Real-time Transport Protocol](/glossary/s/srtp). |
| Use as Fax | Indicate that this SIP account will be used for a fax machine. This will force some settings. |
| Mailbox | Voicemail inbox. |



### Add account

You can add as many accounts as you needed.

To add a account:

* Click the **Add** button

* Enter a **Name** for the account

* Enter values as needed.

* Click **Apply**
