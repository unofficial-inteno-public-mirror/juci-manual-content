

# WiFi encryption

WiFi encryption means to encrypt the messages that are sent between nodes on a wireless network.

A [wifi interface](/glossary/w/wifi_interface) can use one of several encryption options:

| None | No encryption. |
| WEP | [Wired Equivalent Privacy](/glossary/w/wep) |
| WPA2 Personal (PSK) | [Wi-Fi Protected Access](/glossary/w/wpa_personal) |
| WPA/WPA2 Personal (PSK) Mixed Mode | [Wi-Fi Protected Access II](/glossary/w/wpa2_personal) [Pre-Shared Key](/glossary/p/psk) |
| WPA2 Enterprise | [Wi-Fi Protected Access II Enterprise](/glossary/w/wpa2_enterprise) |
| WPA/WPA2 Enterprise Mixed Mode | [Wi-Fi Protected Access Enterprise](/glossary/w/wpa_enterprise) / [Wi-Fi Protected Access II Enterprise](/glossary/w/wpa2_enterprise) |
