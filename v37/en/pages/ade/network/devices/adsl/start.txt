# ADSL

The **ADSL** view allows you to configure [ADSL](/glossary/a/adsl) devices.

##

At the top of the page is a list of selectable devices.

When a particular device is selected, details about it is shown in the configuration section.


^ Section ^ Description ^
| Name | Name of the device. |
| VPI | ATM [Virtual Path Identifier](/glossary/v/vpi). |
| VCI | ATM [Virtual Channel Identifier](/glossary/v/vci). |
| DSL Link Type | [EoA](/glossary/e/eoa) / [PPPoE](/glossary/p/pppoe) / [IPoE](/glossary/i/ipoe). |
| Encapsulation Mode | LLC [SNAP](/glossary/s/snap) / [VC-MUX](/glossary/v/vc_mux).  |
| Service Type | [Service Type](/glossary/s/service_type). |
| Bridge | Setting to enable [network bridge](/glossary/n/network_bridge) use. |
