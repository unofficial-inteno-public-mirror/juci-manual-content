# Create New Network Interface

The **create new network interface**  dialog allows you to create a new [interface](/glossary/n/network_interface) according to your needs.

##

The dialog consists of several sections.

^ Section ^ Description ^
| Interface Name | Name for the interface. |
| Protocol | Select [interface type](/glossary/i/interface_protocol). |
| Interface Type | Select [interface type](/glossary/i/interface_type). |
| Add network to a firewall zone | Connects interface to [firewall zone](../../network/firewall/zones/start). |
